<p>목록 태그를 사용할때 ol과 ul은 부모이기 때문에, 자식으로는 li만 포함될 수 있다. 조심해야하는게 생각없이 내용이라서 p나, 제목 h문을 넣으면 안된다는걸 유의해야한다.</p>
